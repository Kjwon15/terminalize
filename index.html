Terminalize

write html like below. This can be converted to terminal-like UI.


    <ul id="terminal">
      <li><a href="https://kjwon15.net/wordpress/">blog</a></li>
      <li><a href="https://github.com/Kjwon15/terminalize/">sourcecode</a></li>
      <li>
        <h1>subdirectory</h1>
        <ul>
          <li><h1>text</h1>
            the cake is a lie
          </li>
        </ul>
      </li>
    </ul>
  • <ul></ul>
    to make the root directory.
  • <li>
     <h1>filename</h1>
     content
    </li>
    to make a regular file.
  • <li>
     <h1>directory name</h1>
     <ul>
     </ul>
    </li>
    to make a directory.
  • <li>
    <a href="location">link name</a>
    </li>
    to make a link (it works like a directory. cd name to move page).
